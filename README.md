# 星际迷航：伟大之触 (Star Trek: A Touch of Greatness)

《星际迷航：伟大之触》是一部星际迷航同人小说，由约翰·埃里克·埃格 (John Erik Ege) 创作。

## 故事简介

故事发生在 24 世纪末期，讲述了伦纳德·麦考伊上将在 115 岁高龄时的一段非凡旅程。当古伊南带着一个拥有柯克船长、斯波克大使和麦考伊本人基因的孩子出现在他面前时，麦考伊被迫卷入了一场凯尔文人内战的阴谋之中。

凯尔文人是一个高度先进的外星种族，能够在人类形态和原始的触手生物形态之间转换。他们正面临着一场内战，一方希望保持人类形态，另一方则希望回归传统的凯尔文形态。而这个名叫裘德（后改名为塔马斯·帕金·阿巴斯特）的孩子，是唯一一个在产前成功印记凯尔文人心理并幸存下来的人类。

为了确保孩子的安全，麦考伊将他带到了奥兰星——法布林人的殖民地，由他的继女娜塔莉亚和她的家人照顾。

## 主要角色

- **伦纳德·H·麦考伊** - 星际舰队上将，前企业号首席医疗官
- **古伊南** - 酒吧老板，神秘的外星种族成员
- **塔马斯·帕金·阿巴斯特（裘德）** - 拥有多位企业号船员基因的少年
- **凯琳达** - 凯尔文人改良派领袖
- **洛蕾娜** - 塔马斯的生母，凯琳达的后代
- **娜塔莉亚** - 麦考伊的继女，法布林人大祭司之女
- **胡安·加西亚** - 娜塔莉亚的丈夫，质量复制器系统总经理
- **Star** - 基因改造的海豚，海洋生物学家
