## 翻译规范

### 人名翻译
- Leonard H. McCoy -> 伦纳德·H·麦考伊
- James T. Kirk -> 詹姆斯·T·柯克
- Spock -> 斯波克
- Guinan -> 古伊南
- Kelinda -> 凯琳达
- Lorena -> 洛蕾娜
- Jude -> 裘德

### 术语翻译
- Federation -> 联邦
- Starfleet -> 星际舰队
- Kelvan -> 凯尔文人
- Vulcan -> 瓦肯人
- Transporter -> 传送器
- Tricorder ->  tricorder/三录仪

### 文件格式

每章单独保存为一个 Markdown 文件：
- `chapters/00-prologue.md` - 序言
- `chapters/01-chapter-one.md` - 第一章
- ...
