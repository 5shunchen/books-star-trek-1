## 翻译说明

### 专有名词翻译规范

- **人名：**
  - Leonard H. McCoy → 伦纳德·H·麦考伊
  - James T. Kirk → 詹姆斯·T·柯克
  - Spock → 斯波克
  - Guinan → 古伊南
  - Kelinda → 凯琳达
  - Lorena → 洛蕾娜
  - Jude/Tammas → 裘德/塔马斯

- **种族/物种：**
  - Kelvan → 凯尔文人
  - Vulcan → 瓦肯人
  - Caitian → 凯蒂安人
  - Fabrin/Fabrini → 法布林人
  - Dolphin → 海豚（智慧海豚种族）

- **地点/飞船：**
  - Enterprise NCC-1701 → 进取号 NCC-1701
  - K7 → K7空间站
  - Yonada → 约纳达号
  - New Fabrina → 新法布里纳星
  - Oran → 奥兰星

- **技术术语：**
  - Tricorder → 三录仪
  - PADD → PADD（个人访问显示设备）
  - Transporter → 传送器
  - Replicator → 复制器
  - Phaser → 相位枪

### 翻译原则

1. **文学性与可读性并重**：采用流畅优美的中文散文体，保持原文的叙事节奏和情感张力
2. **星际迷航术语规范**：系列特有术语采用粉丝圈公认的译法
3. **文化适应性**：将英文文化隐喻转化为中文读者易于理解的表达
4. **对话口语化**：保持人物对话的口语化特征，符合人物性格
5. **首次出现注释**：对部分英文术语在首次出现时提供中文解释
